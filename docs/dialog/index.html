Self Stacking Dialogs
Opening multiple dialogs will automatically suspend previous dialog, while current dialog is active. Once current dialog is dismissed, previous dialog is returned to focus.


        var dialog = new FlowUI.Dialog({
            title: "Dialog #" + count,
            html: "Clicking 'Open New Dialog' will open another dialog and set current dialog to Inactive. Current dialog will become active again once new FlowUI.Dialog is closed.",
            animation: {
                in: "pulseIn",
                out: "pulseOut"
            },

            buttons: [
                {title: 'Open New Dialog', onclick: function() {
                    demo.multiple(++count);
                }},
                {title: 'Close Current Dialog', onclick: function() {
                    dialog.close();
                }}
            ]
        });
