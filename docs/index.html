Bower

If you're using Bower to manage your packages, you can easily include FlowUI in your project by issuing the following terminal command:


> bower install flowui

Node Package Manager

FlowUI can also be installed if you're using NPM. To add FlowUI to your project, issue the following terminal command:


> npm install flowui
